# microcms-schema

microCMS を利用した制作を支援するツール群

## 概要

このプロジェクトは、microCMS を使った開発をより効率的に行うための支援ツールです。API スキーマの管理や自動化を通じて、開発者の生産性を向上させることを目的としています。

## 主な機能

### 1. API スキーマの解析

microCMS の API スキーマを解析し、構造を可視化・理解しやすくします。

### 2. API スキーマのスキーマ作成

microCMS の API スキーマ設定画面からダウンロードできるスキーマファイル（JSON）の構造を定義するスキーマ（JSON Schema や TypeScript の型定義など）を作成します。これにより、スキーマファイルのバリデーションや型安全な操作が可能になります。

### 3. API スキーマやカスタムフィールドの自動入力

繰り返し作業となる API スキーマやカスタムフィールドの設定を自動化し、作業を効率化します。
